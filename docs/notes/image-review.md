# Luna 图片评审记录 v0.1

这里用于记录每一轮 AI 生图结果的选择、问题和下一轮调整方向。

## 评审方式

每轮生成图片后，可以按下面维度简单打分或写备注：

- 是否像宝宝成长陪伴助手。
- 是否有温柔、可靠、亲切的小姐姐感。
- 家长看到是否会信任。
- 宝宝看到是否愿意互动。
- 手机小屏幕上脸和表情是否清楚。
- 图片比例和分辨率是否符合用途。
- 是否方便扩展成多个表情。
- 是否适合未来放到树莓派小屏幕上。

当前采用图片：

规格检查：

- 表情图可以先用于 v0.1 网页验证。
- 当前图片格式混用了 `png` 和 `jpeg`，后续进入网页 Demo 前建议统一为 `png` 或 `webp`。
- 角色构图和脸部位置接近，但仍可能有轻微跳动，网页验证时需要观察切换效果。
- 缺少独立的 `comforting` 安抚关心表情，v0.1 暂时复用 `listening`。

优点：

- 角色一致性基本可用。
- 表情区分度已经足够支撑第一版对话验证。
- 画风柔和，适合宝宝陪伴方向。

问题：

- 表情数量暂时只有 4 个。
- 图片格式和命名还没有完全统一。
- 部分表情带有手势，切换时可能产生明显位置变化。

下一轮调整：

- 进入网页 Demo，先验证实际切换效果。
- 如果切换跳动明显，再统一裁切尺寸和角色位置。
- 后续补一张独立的 `comforting` 安抚关心表情。
